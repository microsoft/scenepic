What is ScenePic?

All platforms have good support for 2D images, with well-recognized formats such as PNG and JPEG that can be viewed out of the box (no installation) and shared trivially. However, while many formats exist for 3D data, none are well-supported without installation of tools such as MeshLab, Blender, etc.

ScenePic was created for 3D computer vision researchers such as those in Mixed Reality, designed as to be a lightweight, reuseable 3D visualization library. Our goals are to make:

  • Experimentation with 3D data nearly effortless
  • Creation of 3D data incredibly easy
  • Sharing of 3D data happen via a single file, with no extra software required
  • Experience of 3D data performant and built on open standards (HTML, JS, WebGL)
